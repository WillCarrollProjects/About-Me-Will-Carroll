# About Me — Will Carroll

A simple personal "About Me" web page, built as my first hands-on web project.

## What's in this repo

- **`index.html`** — the web page itself (headline, photo, intro paragraphs, and links).
- **`profile.jpg`** — my profile photo.

## About the page

The page introduces me and a few things I love — hiking, skiing, and basketball — along with links to a couple of favorite spots.

## How to view it

Download or clone the repo, then open `index.html` in any web browser.

## Built with

Plain HTML and CSS, created while learning the basics of the terminal, Git, and GitHub.
