# About Me — Will Carroll

This is a personal "About Me" web page made with HTML, CSS, and JavaScript — it shows my name, photo, a few things I love, and some links. It also has interactive buttons: one changes the heading color, one shows a random fun fact about me, and one counts how many times it's been clicked. I built it as my very first web project while learning the terminal, Git, GitHub, and JavaScript with the help of Claude Code. To view it, just open `index.html` in any web browser.
